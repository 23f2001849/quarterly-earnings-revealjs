Evaluating Long-Term Initiatives – NPV

We assess major technology & product investments using NPV:

\[ \text{NPV} = \sum_{t=1}^{T} \frac{CF_t}{(1 + r)^t} - C_0 \]

Return on Equity (ROE) is also tracked quarterly: \[ \text{ROE} = \frac{\text{Net Income}}{\text{Average Shareholders' Equity}} \]

  • \( CF_t \): projected cash flow in period \( t \)
  • \( r \): discount rate (cost of capital / hurdle rate)
  • \( C_0 \): initial investment in the initiative

Interactive Earnings Dashboard – API Sample

Data for the internal earnings dashboard is exposed over a secured API: 


async function fetchQuarterlyEarnings(quarter) {
  const response = await fetch(
    `https://api.examplebank.com/earnings?quarter=${quarter}`,
    {
      headers: {
        Authorization: `Bearer <ACCESS_TOKEN>`,
        "X-Client-Id": "earnings-analytics-ui"
      }
    }
  );

  if (!response.ok) {
    throw new Error(`HTTP ${response.status}: ${response.statusText}`);
  }

  const data = await response.json();

  // Aggregate by business unit for visualization
  return data.businessUnits.map(unit => ({
    name: unit.name,
    revenue: unit.revenue,
    netIncome: unit.netIncome,
    roe: unit.roe
  }));
}
Clarify that this is illustrative pseudo-code, not a production endpoint. Use it to show that the earnings dashboard is API-driven, making it easy to build interactive internal tools and self-service analytics.
